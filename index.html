🔧 Troubleshooting Setup Errors

Sometimes the installer fails on Windows 11. Use these steps to fix common issues:

Antivirus Interference

Your security software might block the setup file. Turn off your antivirus protection for five minutes while you run the installer. Remember to turn it back on once the software installs.

Clear Temporary Files

A failed install can leave behind broken files. Press the Windows key + R on your keyboard. Type "%temp%" and press Enter. Delete all folders and files inside this window. Try the setup process again after you finish this step.

Run in Compatibility Mode

If the app refuses to open, right-click the icon on your desktop. Choose Properties. Click the Compatibility tab. Check the box for "Run this program in compatibility mode for" and select Windows 8 or Windows 10.
